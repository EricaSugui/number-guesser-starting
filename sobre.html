Number Guesser!

O Adivinhador de Números


Sobre esse jogo

Desafio disponível no path de jogos Javascript do site codecademy.
Consiste em fazer a lógica do jogo, em javascript, onde em uma disputa entre o jogador e o computador, vence a disputa quem der o número mais próximo. Em caso de empate, o jogador vence.

Código

Código disponível no GitHub, em https://github.com/EricaSugui/number-guesser-starting.
Esse jogo foi construído usando o fluxo de Deploy Contínuo: escrito em localhost, fazendo push para a master no GitHub, e push para o deploy "produção" no meu servidor HostGator.
Mais detalhes sobre bibliotecas e linguagens no arquivo README.md do respositório.
